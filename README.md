# กระบวนการพัฒนา Web Application ภาครัฐ โดยเจ้าหน้าที่ของรัฐ

เว็บแอปภาครัฐที่เจ้าหน้าที่ของรัฐพัฒนาเอง เน้นที่ความคุ้มค่าและเกิดประโยชน์ต่อประชาชน หรือต่อการปฏิบัติงานเพื่อการบริการประชาชน ตามหน้าที่รับผิดชอบ ซึ่งมีขั้นตอนการพัฒนา ดังนี้ โดยก่อนอื่นต้องเตรียมอุปกรณ์ที่จำเป็น 
1. Server ขออนุมัติใช้ Cloud ภายใต้โครงการ GDDC ของกระทรวง DE เพื่อทำเป็น Web Server
2. ลงระบบปฏิบัติการ Window และโปรแกรม Antivirus
3. ลงโปรแกรม Web Server
4. ลงโปรแกรม Notpad++ 
เมื่อมีอุปกรณ์พร้อมแล้ว มีความรู้เกี่ยวกับภาษา HTML/CSS/JavaScript/PHP จึงเริ่มกระบวนการได้ง่าย
