# กระบวนการพัฒนา Web Application ภาครัฐ โดยเจ้าหน้าที่ของรัฐ

แอปภาครัฐที่เจ้าหน้าที่ของรัฐพัฒนาเอง ไม่จำเป็นต้องยิ่งใหญ่ แต่ควรมีประโยชน์ต่อประชาชนหรือต่อการปฏิบัติงานเพื่อการบริการประชาชนในหน้าที่รับผิดชอบ ซึ่งมีขั้นตอนการพัฒนา ดังนี้

## 1. วิเคราะห์ปัญหา
- เลือกกระบวนงาน
- จำแนกขั้นตอน
- คิดหาแนวทางการประยุกต์ใช้เทคโนโลยีในแต่ะขั้นตอน
> [!WARNING
> เน้น WIN-WIN ประชาชนสะดวก เจ้าหน้าที่สบาย ระบบจึงจะยั่งยืน
